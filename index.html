The final model successfully classified and segmented brain tumors, achieving high accuracy in distinguishing between different tumor types.

Anime Rating Prediction Model

I developed a predictive model to identify the key factors influencing anime ratings. The dataset included features such as genre, episode count, viewership, and studio collaborations. I performed Exploratory Data Analysis (EDA) and found that anime genres like Drama and Romance had higher ratings and viewership, while Comedy had the lowest ratings. After treating missing values with median imputation and creating new features (e.g., years_running), I prepared the data by encoding categorical features using One Hot Encoding. A Linear Regression model was developed using statsmodels to predict the rating of an anime, providing valuable insights into the factors that impact user ratings.
